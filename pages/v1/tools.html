Local Downloaders

download-mods.sh

Bash script that downloads all 291 mods in parallel.

chmod +x download-mods.sh && ./download-mods.sh

Location: /Users/steven/CoX-mod-Adventure/download-mods.sh

download-mods.py

Python script with configurable download concurrency.

python3 download-mods.py

Location: /Users/steven/CoX-mod-Adventure/download-mods.py

Installation Guide

1. Download Mods

Run either the bash or python downloader to fetch all mods to the Mods directory.

2. Locate Game Directory

~/Library/Application Support/LaunchCat/coh/assets/mods/

3. Install Mods

Extract zip files into the mods directory. The paths are already configured in LaunchCat's launch line.

4. Launch Game

Start LaunchCat. Mods are loaded automatically with -assetpath assets/mods.
